# Sky Blaster 开发日志

## 项目概述

**项目名称**：Sky Blaster（天空突击者）
**项目类型**：2D竖版卷轴射击游戏 + Roguelike技能升级系统
**目标用户**：7-8岁男孩
**技术栈**：HTML5 Canvas + 原生JavaScript（单文件）
**项目路径**：`/home/sudong/mimo code/sky-blaster/`

## 开发时间线

### 第一阶段：项目规划与设计（2026-06-16）

**需求确认**：
- 游戏类型：2D竖版飞行射击
- 视觉风格：像素风
- 核心玩法：自动射击 + Roguelike技能升级
- 难度系统：3种难度可选
- 目标用户：7-8岁男孩

**设计决策**：
- 技术选型：HTML5 Canvas（浏览器直接运行）
- 架构：模块化JS文件 → 后改为单文件（解决CORS问题）
- 美术：程序化像素画精灵（无需外部图片）

### 第二阶段：核心开发（2026-06-16）

**完成的功能**：
1. 游戏循环与状态管理
2. 玩家飞船控制系统
3. 4种敌人类型（侦察机、战斗机、轰炸机、无人机）
4. Boss系统（3关各1个Boss，多阶段攻击）
5. 子弹系统与碰撞检测
6. 经验球与升级系统
7. 8种可升级技能
8. 粒子特效系统
9. 音效系统（Web Audio API）
10. 完整UI系统（主菜单、暂停、游戏结束、胜利）

### 第三阶段：问题修复与优化（2026-06-16）

**遇到的问题与解决方案**：

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 黑屏无法运行 | ES6模块需要服务器 | 创建单文件play.html |
| 子弹不发射 | 缺少active:true属性 | 为所有子弹添加active属性 |
| Boss不掉血 | DIFF配置缺少bossHp | 添加bossHp属性到难度配置 |
| 追踪弹失效 | 只追踪enemies数组 | 添加Boss作为追踪目标 |
| 关卡过渡卡死 | 游戏循环不调用levelUp状态 | 修改循环条件包含levelUp |
| 经验球吃不到 | 碰撞检测使用角坐标 | 改为中心点距离检测 |
| 精灵图全黑 | 颜色代码无效 | 添加颜色映射表 |
| 无敌帧太闪 | 频繁切换可见性 | 改为绿色护盾光环效果 |

### 第四阶段：视觉优化（2026-06-16）

**改进内容**：
- 分辨率从240x360提升到960x1440
